Trainings

Computer System Servicing NC II - Core Competencies

A comprehensive TESDA program divided into four competency areas (COC 1–4).

🖥️ COC 1: Installing and Configuring Computer Systems
• Assemble and disassemble PCs
• Install operating systems (Windows/Linux)
• Configure BIOS, drivers, and software
• Tools: Screwdrivers, ESD strap, USB installer, diagnostic tools

🌐 COC 2: Setting Up Computer Networks
• Create UTP cables using crimping tools
• Configure LAN with routers/switches
• Assign static/DHCP IPs, enable file & printer sharing
• Tools: Crimping tool, cable tester, punch-down tool, router, switch

🖧 COC 3: Setting Up Computer Servers
• Install Windows Server or Linux Server
• Create user accounts and assign permissions
• Manage shared folders, DHCP/DNS configuration
• Tools: Server OS media, Active Directory, RDP, Server Manager

🛠️ COC 4: Maintaining and Repairing Computer Systems and Networks
• Perform diagnostics and preventive maintenance
• Replace defective components (RAM, HDD, PSU)
• Virus removal, backup and recovery
• Tools: Multimeter, cleaning kit, antivirus tools, software utilities
